Protocol Support Matrix
📡
MQTT v3.1 / v5.0
Supported
Primary ingestion via Eclipse Mosquitto. Supports QoS 0/1/2, retained messages, and LWT for device health monitoring.
Compatible: Advantech ADAM, Siemens IOT2040, any MQTT-enabled gateway
🔗
OPC-UA (IEC 62541)
Supported
node-opcua client connects to S7-1500, B&R PLCs. Subscription-based (not polling) for real-time data efficiency.
Compatible: Siemens S7-1200/1500, B&R Automation Studio, Kepware OPC-UA
🔌
EtherNet/IP (CIP)
Via Kepware
Rockwell ControlLogix / CompactLogix via Kepware KEPServerEX as EtherNet/IP-to-OPC-UA bridge. Native driver also available.
Compatible: Allen-Bradley ControlLogix 5580, CompactLogix 5380, MicroLogix, PowerFlex
📶
Modbus TCP / RTU
Supported
node-red-contrib-modbus supports TCP and RTU. Energy meters, water meters, and legacy drives via RS-485/USB converter.
Compatible: Schneider iEM3000/3100, ABB DMTME, Yokogawa analyzers, VACON drives
🏛️
FactoryTalk Historian
Roadmap Q3
Integration via FT Historian SDK or PI Web API bridge. Pulls historical context into TimescaleDB to supplement live PLC data.
Compatible: FactoryTalk Historian SE, PI Server, FactoryTalk AssetCentre
☁️
Google Cloud Pub/Sub
Config Ready
Set DATA_SINK=gcp in .env to activate. All 119 fields include tenant_id, plant_id, device_id for BigQuery partitioning.
Targets: BigQuery, Cloud Spanner, Looker, Vertex AI
