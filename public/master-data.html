📖 Cycle Time Reference

Ideal Cycle Time is the theoretical minimum time to produce one part with zero losses. Used as the denominator in OEE Performance calculation.

OEE Performance = (Ideal Cycle Time × Parts Produced) ÷ Run Time

Target Parts / Shift is auto-calculated: (43200 sec shift ÷ Ideal Cycle) × typical availability (≈ 90%)

💡 Demo Machine Hobbing Machine (896) — cycle times range from 36s (DH102208 66T) to 255s (F165111 36T) depending on tooth count and module. Development programs have no cycle time set until engineering qualification is complete.
